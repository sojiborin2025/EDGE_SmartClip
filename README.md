# SmartClip

**SmartClip** is a lightweight clipboard enhancement extension designed for Edge/Chrome browsers. It significantly streamlines the repetitive tasks of extracting, cleaning, and formatting specific text patterns (such as SCN IDs or Markdown content) through intelligent detection and chained action logic.

### ğŸš€ Key Features
*   **Double-Ctrl+C Trigger:** Seamlessly integrates with your habits. The functional menu only appears upon pressing `Ctrl+C` twice, ensuring zero interference with normal browsing.
*   **Smart Extraction:** Precisely captures target IDs like `[SCN-XX-XX]` from messy sentences, eliminating the need for tedious mouse selection.
*   **Chained Workflow:** Supports "Extract then Transform" logic, allowing complex text processing in a single, fluid session.
*   **Optimized UX:** Features keyboard shortcuts (Number keys 1-5) and `Space` key for repeating the last action, enabling a high-speed, keyboard-driven experience.


### ğŸ› ï¸ Functionalities
1.  **Smart Capture:** One-click extraction of identifiers within the selected range.
2.  **Format Cleaning:** Automatically removes redundant Markdown symbols (##, **, etc.).
3.  **Wrapping & Conversion:** Quickly add quotes or wrap text into ` ```TXT ``` ` code blocks.
4.  **Live Preview:** Instantly displays the processing object to ensure accuracy.

**Installation:**
This tool is currently shared as an open-source project and is not listed on official app stores. To use it, please download this repository, enable "Developer mode" in Edge, and install it manually via the "Load unpacked" option.
